# 记忆闪光

一个适合儿童玩的单文件 HTML 记忆益智小游戏，玩法类似 Simon。玩家需要记住颜色闪光顺序，并在 5 秒内按顺序点击对应按钮。游戏支持每 5 个按键升级一次的欢呼动画，并会在本机保存最长的 10 次闯关记录。

## 运行方式

直接用浏览器打开 `index.html` 即可游玩，不需要安装依赖，也不需要启动服务器。

## 发布到 GitHub Pages

把仓库推送到 GitHub 后，可以在仓库的 `Settings` -> `Pages` 中选择 `Deploy from a branch`，分支选择 `main`，目录选择 `/root`。保存后 GitHub 会生成一个可分享的网页链接。
